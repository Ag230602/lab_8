
Lab 8 Individual Submission – Domain Adaptation for GenAI Systems

Files Included
--------------
1. Individual_Contribution_Report.pdf
2. instruction_dataset.json
3. finetune_lora.py

Description
-----------
This package demonstrates the individual work performed for Lab 8,
including instruction dataset creation and LoRA-based fine-tuning
for a domain-adapted disaster forecasting assistant.

Usage
-----
1. Install dependencies:
pip install transformers peft datasets trl

2. Run fine-tuning:
python finetune_lora.py
